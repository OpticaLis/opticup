📦 הכנסת מלאי

  1. הכנסה ידנית: מלא את פרטי המסגרת ← לחץ "צור ברקודים" ← הורד Excel לברקודים ← הדפס את הברקודים ← רק אחרי ההדפסה לחץ "הכנס למלאי"
  2. הכנסה מאקסל: העלה קובץ Excel עם עמודות: מותג, דגם, גודל, צבע, כמות, מחיר עלות ← אשר ייבוא ← פריטים מוכרים נכנסים אוטומטית ← פריטים חדשים ממתינים לברקוד ← לחץ "צור ברקודים" ← הדפס ← אשר
  3. קבלת סחורה: בחר ספק ← אופציונלי: קשר להזמנת רכש ← הוסף פריטים ← אשר קבלה

📤 הורדת מלאי

  1. הורדה ידנית: חפש לפי ברקוד או בחר מותג ← דגם ← גודל ← צבע ← לחץ "הורדה מהמלאי" ← בחר סיבה ← הזן PIN
  2. הורדה מאקסל: העלה קובץ Excel עם עמודות ברקוד וכמות

📋 הזמנות רכש

  1. לחץ "+ הזמנה חדשה" ← בחר ספק ← הוסף פריטים (ברקוד / ידני) ← שמור טיוטה
  2. כשמוכן לשלוח: לחץ "שלח" ← ההזמנה עוברת לסטטוס "נשלחה"
  3. כשהסחורה מגיעה: פתח קבלת סחורה ← קשר ל-PO ← אשר קבלה
  4. לאחר קבלה מלאה: לחץ "קלוט למלאי" ב-PO

🏷️ ברקודים — כללי זהב

  • ⚠️ אסור להכניס מסגרת למלאי ללא ברקוד מודפס ומודבק
  • פורמט ברקוד: 2 ספרות סניף + 5 ספרות רצות (לדוגמה: 0000123)
  • מסגרת קיימת במלאי — מקבלת את הברקוד הקיים שלה
  • מסגרת חדשה — מקבלת ברקוד חדש בסדר עולה

הערות חשובות:

  • מותג — חייב להיות קיים במערכת
  • כמות — מספר שלם, מינימום 1
  • מחיר עלות / מכירה — מספר, ניתן להשאיר ריק
  • סינכרון — מלא / תדמית / לא (אם ריק — ברירת מחדל: מלא)
  • פריטים חדשים ללא ברקוד — יועברו לרשימת המתנה לברקוד
  • פריטים קיימים עם ברקוד — כמות תתעדכן אוטומטית
