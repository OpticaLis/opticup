# אופטיקה פריזמה — מערכת מלאי

## תיאור
מערכת ניהול מלאי למשקפיים ואביזרי אופטיקה.
בנוי עם: HTML/JS/CSS + Supabase + SheetJS

## מודולים
- **מלאי ראשי** — צפייה, סינון, עריכה, ייצוא Excel
- **הכנסת מלאי** — ידנית / מהזמנת רכש / ייבוא מאקסל
- **הזמנת רכש** — יצירה ומעקב
- **ניהול מותגים וספקים**

## לוגיקת ברקוד
פורמט: `BBDDDDD` (2 ספרות סניף + 5 ספרות עולות)
- מקסימום 99 סניפים (00–99)
- מקסימום 99,999 פריטים לסניף
- ברקוד שנמכר לא חוזר לעולם
- ברקוד כפול → שגיאה בעברית
