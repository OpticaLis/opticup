איך זה עובד
  • 4 בורדים = 4 lanes אופקיים, כל אחד צבע משלו. עין אחת רואה את כל מערך האוטומציה.
  • כל חוק = שורה: trigger node → arrow → action node. בורד עם הרבה חוקים (אירועים, 8) — מסודרים grid צפוף.
  • לחיצה על node פותחת panel עריכה (slide-in מימין). שינוי מתעדכן ב-DB מיד.
  • "+ חוק חדש" בכל lane — drag a node from a palette, או modal mini-wizard.
  • ביצירה: stack מותר ("שלח SMS, ואחר כך תוסיף לתור reminder ב-2 ימים") — נדחה לשלב מאוחר. כרגע node trigger → node action בלבד.
  • חוק כבוי = node שקוף, אייקון ⏸ במקום ⚡.
⚠️ דורש כתיבה רבה: הצגה ב-grid לא טריוויאלית, slide-in panels, drag-drop אופציונלי. הכי כבד מבין השלוש.
