============================================
Optic Up — שירות סנכרון Access
============================================

דרישות:
* Node.js (גרסה 18+) — הורד מ: https://nodejs.org
* מחשב Windows מחובר לאינטרנט

התקנה:
1. העתק את כל התיקייה הזו למחשב (למשל: C:\OpticUp-Watcher)
2. לחיצה כפולה על setup.bat
3. עקוב אחרי ההוראות על המסך
4. זהו\! השירות רץ ברקע אוטומטית.

הסרה:
* לחיצה כפולה על uninstall.bat

בדיקה:
* פתח inventory.html בדפדפן
* לך לטאב "סנכרון Access"
* צריך להופיע: Watcher פעיל (ירוק)

מבנה תיקיות Dropbox:
InventorySync/
├── sales/      ← מכירות מ-Access נכנסות לכאן
├── new/        ← מלאי חדש מ-Optic Up יוצא לכאן
├── processed/  ← קבצים שעובדו
└── failed/     ← קבצים שנכשלו
